Second monitor display

Shows the vector plot full-screen on a second monitor, while the main PIVlab window keeps the controls — useful for presentations or a dedicated display wall.

Experimental PIVlab labels this feature "experimental" in the menu itself. It's under active refinement, so expect rough edges.

Open via Plot → Second monitor display (experimental) (no keyboard shortcut). Toggling it on opens a full-screen black window on your second monitor and mirrors the vector plot there instead of the main window; the main window shows the PIVlab logo in its place. Toggling it off (from the same menu item, or by closing the second-monitor window) restores the normal single-window display.

Needs an actual second monitor If only one display is detected, PIVlab still opens a second window, but on your only monitor — there's no benefit without genuinely separate screens.

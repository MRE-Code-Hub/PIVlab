Stereo PIV settings

Stereo PIV (two-camera, 3-component velocity measurement) is not implemented yet. This page exists so the menu structure is documented honestly rather than left silently unexplained.

Not currently functional Analysis → Stereo PIV settings → Disparity correction is present in the menu but disabled, and the parent menu item currently does nothing when clicked. There is no working stereo-PIV analysis path in this version.

Some related, already-usable infrastructure exists elsewhere and is worth knowing about if you're setting up a two-camera rig for future use:

  • The Stereo-PIV mode (2D3C) checkbox exists in the Input data panel but is currently disabled.
  • Camera calibration already supports a Scheimpflug adapter (tilted sensor model) option, relevant to angled stereo camera setups, and its "Camera 2" menu entries exist (though disabled today) for future dual-camera support.

If you need stereo PIV today, this isn't yet the tool for it — check the PIVlab website for news on when this feature lands.

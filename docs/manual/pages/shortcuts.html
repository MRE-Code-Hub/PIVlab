Keyboard shortcuts

Every shortcut jumps straight to a panel — the same panel you'd reach by clicking the matching menu item. There's no need to memorize menu paths once you know these.

Ctrl+VVelocity-based vector validation

Ctrl+TMeasure distance and angle / markers
Ctrl+PExtract parameters from poly-line
Ctrl+QExtract parameters from area
Ctrl+BStatistics
Ctrl+GSynthetic particle image generation
Ctrl+HGetting-started manual
Masking has no shortcut PIVlab's printable shortcuts card (Learn! → List keyboard shortcuts) lists Ctrl+E as covering both "region of interest" and "mask" — that's left over from when the two lived in one panel. In the current version, Ctrl+E opens only the ROI panel; masking has moved to its own panel, reachable via Image settings → Define masks or the quick-access toolbar icon, with no dedicated keyboard shortcut.

Everything not listed above (loading/saving sessions, camera calibration, exports, stereo settings, streamlines, correlation matrices, and so on) is reached through the menu only.

Printable copy Learn! → List keyboard shortcuts opens a one-page PDF version of this list you can print or pin next to your desk.

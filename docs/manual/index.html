PIVlab Manual

PIVlab is free, open-source particle image velocimetry (PIV) software, available as a standalone application — no MATLAB license required — or as a MATLAB toolbox. This manual explains how to use the graphical interface, one task at a time.

Use the sidebar to browse chapters, press / to search, and use the “On this page” panel to jump between sections.
